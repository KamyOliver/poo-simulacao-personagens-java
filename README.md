**Simulação de Personagens em Java**

Projeto em Java que simula personagens executando ações automaticamente, como caçar, comer e dormir, utilizando conceitos de Programação Orientada a Objetos (POO).

**Funcionalidades**
Execução de ações automáticas (caçar, comer, dormir)
Simulação com comportamento aleatório
Múltiplos personagens
Exibição do estado dos personagens
Evolução do projeto em versões (V1, V2 e V3)

**Conceitos aplicados**
JogoV1.java → Versão inicial com ações básicas
JogoV2.java → Simulação com ações aleatórias
JogoV3.java → Simulação com múltiplos personagens
Personagem.java → Classe principal dos personagens
Personagem2.java → Versão alternativa/evoluída da classe Personagem

**Estrutura do projeto**
JogoV1.java → Execução simples de ações do personagem
JogoV2.java → Simulação com ações aleatórias
JogoV3.java → Simulação com múltiplos personagens

**Como executar**
Compile e execute pelo terminal:

javac JogoV1.java
java JogoV1
Ou execute outras versões:
javac JogoV2.java
java JogoV2
javac JogoV3.java
java JogoV3

**Objetivo**
Projeto acadêmico desenvolvido para praticar conceitos de Programação Orientada a Objetos e simulação de comportamento em Java.
