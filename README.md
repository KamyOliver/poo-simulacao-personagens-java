Simulação de Personagens em Java
Este projeto implementa uma simulação simples de personagens que realizam ações automaticamente, como caçar, comer e dormir, utilizando conceitos de programação orientada a objetos em Java.

Funcionalidades
Criação de personagens com atributos
Execução de ações (caçar, comer, dormir)
Simulação automática com eventos aleatórios
Exibição do estado do personagem em tempo real
Uso de múltiplos personagens

Conceitos aplicados
Programação Orientada a Objetos (POO)
Classes e objetos
Construtores
Sobrecarga de construtores
Uso de Random
Estruturas de controle (while, switch)
Threads (Thread.sleep)

Estrutura do projeto
JogoV1.java → Execução simples de ações do personagem
JogoV2.java → Simulação com ações aleatórias
JogoV3.java → Simulação com múltiplos personagens

Como executar
Compile e execute pelo terminal:
javac JogoV1.java
java JogoV1
Ou para versões mais avançadas:

javac JogoV2.java
java JogoV2
javac JogoV3.java
java JogoV3

Objetivo
Projeto acadêmico desenvolvido para praticar conceitos de Programação Orientada a Objetos e simulação de comportamentos com Java.
